### A simple  journal with an ability to alarms for certain tasks. Written in python.

Jouranl saves tasks and their alarm status in sqlite database. Journal allows sorting using date of addition or date of alarm.


Dependencies:
- sqlite3
- tkinter
- pytest (for testing)

Tests will be present inside tests dir. 


**Project was made for unit-testing course.**

***
</br></br>

1. User interface uses tkinter and is written in gui.py
2. Business logic is written in logic.py
3. All work with database is written in database.py
